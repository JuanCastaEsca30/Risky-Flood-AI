Conversión de archivos BSON a GEOJSON
https://www.mongodb.com/es/resources/basics/json-and-bson

--Formato MongoDB--
{
  "_id": "65a1b2c3d4e5f67890123456",
  "nombre": "Río Nautla", 
  "tipo": "corriente",
  "longitud_km": 125,
  "geometry": {
    "type": "LineString",
    "coordinates": [[-97.5, 20.4], [-97.6, 20.3], ...]
  }
}

--FormatoGeoJson que necesita Leaflet
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "id": "65a1b2c3d4e5f67890123456",
      "properties": {
        "nombre": "Río Nautla",
        "tipo": "corriente", 
        "longitud_km": 125
      },
      "geometry": {
        "type": "LineString",
        "coordinates": [[-97.5, 20.4], [-97.6, 20.3], ...]
      }
    }
  ]
}
